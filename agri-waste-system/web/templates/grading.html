看图分级教学（优/良/中/差）

这里提供一套“快速判别”规则：颜色/气味/霉变/杂质/含水率/来源可追溯性。

一分钟分级法（建议顺序）
  1. 先看红线：严重霉变/腐败/异味 → 直接判为“差”（或走无害化兜底）。
  2. 再看可追溯：来源不明 → 避免食品/精细提取路线，等级不宜评“优”。
  3. 再看杂质：塑料/金属/泥沙多 → 降级并优先走“材料/能源/农用”方向。
  4. 最后看新鲜度：存放过久、褐变明显 → 降级并避开高要求提取。
等级对照表
优
新鲜、无霉变异味、来源明确、杂质少，适配高值化路线。
良
轻度瑕疵可控（轻度霉点/轻微褐变），仍可做材料/农用/部分提取。
中
质量波动明显、杂质偏多或存放偏久，优先选择稳妥落地路线（材料/能源/堆肥）。
差
严重霉变腐败/强异味/不可追溯风险高，禁入食品与精细提取，走兜底与安全处置。
